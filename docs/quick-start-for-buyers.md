# Quick Start For Buyers

Use this page to request a small, legal, fixed-scope GitHub task.

## Step 1: Pick One Offer

### Repository Cleanup

Good for a repository that needs basic professional setup.

Includes:

- README structure cleanup
- SECURITY.md
- Pull request template
- Issue template
- Basic `.gitignore` review

### GitHub Actions Fix

Good for one failing workflow.

Includes:

- Root-cause summary
- YAML/config patch
- Validation notes

### Defensive Review Lite

Good for one small PR, module, or file set.

Includes:

- Bug and regression-risk notes
- Test gap summary
- Safe hardening suggestions
- Optional small patch

## Step 2: Prepare Inputs

Please provide:

- Repository link
- Issue, PR, or failing workflow link
- Confirmation that you own or are authorized to request work
- Desired deliverable
- Deadline
- Budget range
- Whether public pull requests are acceptable

## Step 3: Open A Request

Open an issue with the service request template:

`https://github.com/zhangquanshen123-droid/-/issues/new?template=service-request.md`

## Work Boundaries

I do not perform:

- Unauthorized security testing
- Exploit development
- Intrusive scanning
- Credential handling
- Payment-account operation
- Private data extraction

Work pauses if scope, authorization, disclosure, or payment terms are unclear.
